[name::limit-changes]
[SpocWebEntityId::22261]
[TrizPrinciple::[[Triz-Equipotentiality]]]

In a potential field, limit position changes (e.g. change operating conditions to eliminate the need to raise or lower objects in a gravity field).

## Examples: 

- #has_/examples :: Locks in a Channel between 2 Bodies of Water
- #has_/examples :: 'Skillets' in an automobile plant that bring all tools to the right position (also demonstrates Principle 10, Preliminary Action)
- #has_/examples :: Spring loaded Parts Delivery System in a Factory
- #has_/examples :: Automobile Engine Oil is changed by Workers in a Pit to avoid using expensive Lifting Equipment or unsafe Ramps.

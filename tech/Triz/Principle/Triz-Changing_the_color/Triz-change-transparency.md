Change the Transparency of an Object or its external Environment.

## Examples: 

- #has_/examples :: Use Photolithography to change transparent Material to a solid Mask for Semiconductor Processing. Similarly, change mask material from transparent to opaque for silk screen processing.
- #has_/examples :: A transparent Bandage enabling a Wound to be inspected without removing the Dressing.

[name::merge-identical]
[SpocWebEntityId::22249]
[TrizPrinciple::[[Triz-Combining(Integration)]]]

Bring closer together (or merge) identical or similar objects, assemble identical or similar parts to perform parallel operations.

## Examples: 

- #has_/examples :: Electronic chips mounted on both sides of a circuit board or subassembly 
- #has_/examples :: Personal computers in a network
- #has_/examples :: Thousands of microprocessors in a parallel processor computer
- #has_/examples :: Vanes in a ventilation system

[name::merge-in-space]
[SpocWebEntityId::22252]
[TrizPrinciple::[[Triz-Nesting]]]

Place one object inside another; place each object, in turn, inside the other.

## Examples: 

- #has_/examples :: Measuring in Cups or Spoons
- #has_/examples :: Mechanical Pencil with Lead stored inside.
- #has_/examples :: Portable Audio System (microphone fits inside Transmitter, which fits inside Amplifier Case)
- #has_/examples :: Russian Dolls

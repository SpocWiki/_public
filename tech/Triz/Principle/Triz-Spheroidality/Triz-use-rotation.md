[TrizPrinciple::[[Triz-Spheroidality]]]

Go from linear to rotary motion, use centrifugal forces.

## Examples: 

- #has_/examples :: Use spherical Casters instead of cylindrical Wheels to move Furniture.
- #has_/examples :: Replace wringing Clothes to remove Water with spinning Clothes in a Washing Machine.
- #has_/examples :: Produce linear Motion of the Cursor on the Computer Screen using a Mouse or a Trackball.

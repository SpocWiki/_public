[TrizPrinciple::[[Triz-Feedback]]]

Introduce feedback (referring back, cross-checking) to improve a process or action.

- #has_/examples :: Automatic Volume Control in Audio Circuits
- #has_/examples :: Measurements are used to decide when to modify a Process. 
- #has_/examples :: Signal from Gyrocompass is used to control simple Aircraft Autopilots.

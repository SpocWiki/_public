[TrizPrinciple::[[Triz-Periodic_action]]]

If an action is already periodic, change the periodic magnitude or frequency.

## Examples: 

- #has_/examples :: Replace a continuous Siren with Sound that changes Amplitude and Frequency. 
- #has_/examples :: Use Frequency Modulation to convey Information, instead of Morse Code.

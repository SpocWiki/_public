[name::periodic-action]
[SpocWebEntityId::22281]
[TrizPrinciple::[[Triz-Periodic_action]]]

Instead of continuous action, use periodic or pulsating actions.

## Examples: 

- #has_/examples :: A Warning Lamp flashes so that it is even more noticeable than when continuously lit.
- #has_/examples :: Hitting something repeatedly with a hammer.
- #has_/examples :: An Impact Wrench loosens corroded Nuts using Impulses rather than continuous Force.
- #has_/examples :: Use Pulsed Operation instead of continuous to increase Signal to Noise Ratio while not exceeding inert Heat Production(also Principle 21, Rushing through).
- #has_/examples :: Replace a continuous Siren with a pulsed Sound.

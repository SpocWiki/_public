[name::anti-stresses]
[SpocWebEntityId::22257]
[TrizPrinciple::[[Triz-Prior_counteracion]]]

Create beforehand stresses in an object that will oppose known undesirable working stresses later on.

## Examples: 

- #has_/examples :: Masking Anything before harmful Exposure: Use a Lead Apron on Parts of the Body not being exposed to X-rays. Use Masking Tape to protect the Part of an object not being Painted
- #has_/examples :: Pre-stress Rebar before pouring Concrete.
- #has_/examples :: Reinforced Shaft made from several Pipes which have been previously twisted.

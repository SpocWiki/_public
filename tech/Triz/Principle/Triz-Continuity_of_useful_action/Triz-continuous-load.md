[name::continuous-load]
[SpocWebEntityId::22284]
[TrizPrinciple::[[Triz-Continuity_of_useful_action]]]

Carry on work continuously; make all parts of an object work at full load, all the time.

## Examples: 

- #has_/examples :: Run the Bottleneck Operations in a Factory continuously, to reach the optimum Pace. (From Theory of Constraints, or Takt Time Operations)
- #has_/examples :: Flywheel (or hydraulic System) stores Energy when a Vehicle stops, so the Motor can keep running at optimum Power.

- [ #has/_result :: Thing ] or

The result produced in the action. E.g. John wrote &lt;em&gt;a book&lt;/em&gt;.

Relation describes that: 
[ #has_/domain  :: Action ]
( #has_/name :: is_result )
( #has_/range :: Thing )
